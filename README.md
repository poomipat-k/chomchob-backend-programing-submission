# Chomchob backend programming submission
## Tools
 - Backend Node.js (Express.js)
 - Database MySQL host on remotemysql.com
 - javascript MySQL driver -> mysql2

## How to run this api
1. git clone
2. npm install
3. npm run dev
4. Go to localhost:5000
